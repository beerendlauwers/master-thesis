Werk College 8

FP

Some more Tree based exercises

  • Given the type of a tree: 
           data Tree a = Node (Tree a) a (Tree a) | Leaf
    • Write functions which produces a lists containing all the =a='s from this tree in prefix-, infix- and postfix-order
    • We call a tree full if all paths from the root to a leaf have the same length. Compute the largest full subtree, having the same root at the original tree, by replacing Node 's by Leaf 's
    • Write the function levels :: tree a -> a? which returns a list containing lists of nodes which live at the same level (increasing by level)
  • Given the type of a tree 
          data Tree = Node (Tree a) (Tree a) | Leaf a
    • Write a function maxdepth which computes the maximum depth of a tree
    • If you have not already done so use the function foldTree to do so
    • We define the front of a tree as the list of nodes which live a the maximum depth of the tree. Write a function which computes the front of the tree.
    • If you have not done so, write this function using foldTree

    • In case you have not done so write a function which computes the maximum depth and the front at the same time, using a foldTree again.

-- DoaitseSwierstra - 11 Oct 2010
